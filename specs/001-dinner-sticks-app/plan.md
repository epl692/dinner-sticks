## Summary

Dinner Sticks is a cross-platform (Flutter / iOS + Android) offline meal-planning app. Users maintain named pools of meal ideas (sticks), run a weekly random draw of 5 from any pool, and pick tonight's meal from the resulting bin. All data persists locally via Isar (ACID, reactive NoSQL). State management uses Riverpod 2.x with Clean Architecture (3 layers: domain, data, presentation). Tests are written test-first; domain + data coverage в‰Ґ80% is enforced in CI. Development builds run in a reproducible Docker container (Android); iOS builds use GitHub Actions `macos-latest`.
